# 🤝 Contributing Guide

Thank you for your interest in contributing to this repository!

We welcome contributions that improve the project and help other developers prepare for frontend interviews.

---

# How You Can Contribute

You can contribute by:

- Adding interview questions
- Adding machine coding problems
- Improving explanations
- Fixing typos or documentation
- Adding new frontend resources

---

# Contribution Steps

1. Fork the repository

2. Clone your fork

```
git clone https://github.com/YOUR_USERNAME/frontend-interview-prep.git
```

3. Create a new branch

```
git checkout -b feature/new-topic
```

4. Make your changes and commit

```
git commit -m "Added new JavaScript interview question"
```

5. Push to your fork

```
git push origin feature/new-topic
```

6. Open a Pull Request

---

# Contribution Guidelines

Please ensure:

- Code is clean and readable
- Folder structure is followed
- Documentation is clear
- Topics are relevant to frontend interviews

---

# Code of Conduct

Be respectful and constructive when interacting with others.

---

Thank you for contributing and helping make this repository better! 🚀
